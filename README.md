# Bank Management System (C++ Project) 💰

A console-based C++ project to create and manage bank accounts with simple file handling.

## 🛠️ Features
- Create account
- View all accounts
- Stores data in a binary file

## 💻 Technologies Used
- C++
- File handling using fstream
